================================================================================
                    GITHUB CREDENTIALS
================================================================================

Method 1: HTTPS (Easier)
  1. Enter GitHub username
  2. For password, use Personal Access Token (see below)
  3. Generate token: https://github.com/settings/tokens
  4. Scope: select "repo"
  5. Use token as password

Method 2: SSH (More Secure)
  1. Generate key: ssh-keygen -t ed25519
  2. Add to GitHub: https://github.com/settings/keys
  3. Use SSH URL: git@github.com:YOUR_USERNAME/selenium-java-framework.git
