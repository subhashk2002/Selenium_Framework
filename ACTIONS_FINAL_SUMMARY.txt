================================================================================
                      USAGE IN PAGE OBJECTS
================================================================================

Pattern: Import Action Class, Use in Page Object Methods

Example:
--------
package com.automation.pages;

import org.openqa.selenium.By;
import com.automation.actions.CommonActions;
import com.automation.actions.FormActions;

public class LoginPage extends BasePage {

    private static final By USERNAME = By.id("username");
    private static final By PASSWORD = By.id("password");
    private static final By LOGIN_BTN = By.id("loginBtn");

    // Methods use CommonActions
    public void login(String username, String password) {
        CommonActions.clearAndType(USERNAME, username);
        CommonActions.clearAndType(PASSWORD, password);
        CommonActions.clickWithRetry(LOGIN_BTN);
        CommonActions.waitForUrlContains("/dashboard");
    }
}

Each Page Object:
  ✅ Imports appropriate action class
  ✅ Uses action methods in page object methods
  ✅ Keeps page objects clean and focused
  ✅ Delegates to action classes
